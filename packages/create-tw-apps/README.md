## README

![snapshot](https://cdn.jsdelivr.net/gh/oeyoews/neotw@main/img/create-tw-app.png)

* 快速创建一个 tiddlywiki-application(nodejs)

## usage

```bash
# 下面使用 npm 为例, 你也可以使用yarn, 或者pnpm, 或者其他包管理器(bun)
npm create tw-app
# global install
npm i -g create-tw-app
create-tw-app
# 进入你刚才创建的目录
# 启动tiddlywiki
npm run start
# 生成单文件html
npm run build
```
